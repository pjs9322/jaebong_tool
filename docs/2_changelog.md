# 📝 재봉툴 PRO 체인지로그 (Changelog)

버전 및 날짜별 업데이트 결과 요약을 누적 기록하는 문서입니다.

## [2026-03-04] v13 업데이트 내역
**프로젝트 단위 구축 및 코어 업데이트, 함수 및 변수 리팩토링**

### ✨ 주요 변경 사항
1. **PDF 파일 다운로드 개편**
   - PDF 의뢰서 디자인을 공유 링크(웹 뷰어) 모던 카드 UI 디자인과 동일하게 개편 (그림자, 라운딩 박스, 라벨 디자인 통일)
2. **저장되지 않은 작업 보호 및 경고 강화**
   - 작업 중인 요청사항이 있을 때 새로운 캡처/파일을 불러오거나 다른 수정사항 클릭 시 기존 내용을 덮어쓸지 확인하는 `confirm` 경고창 추가
   - 메모 저장하지 않고 하단 "리스트 추가하기" 선택 시 차단(`alert`) 대신 추가할 것인지 묻는 선택지 제공 및 UI 잔상 제거 로직 적용
3. **좀비 데이터 및 히스토리 오염 개선**
   - 이미지를 지우고 텍스트 전용으로 저장 시 이전 이미지의 데이터가 내역에 이월되어 나타나는 현상 해결 (`S.baseImgSrc` 강제 초기화)
4. **튜토리얼 중복 표시 제거**
   - `localStorage`를 활용하여 튜토리얼을 1회 완료/스킵한 기기에서는 재표현되지 않도록 조치 완료
