# Task 002: 공유 페이지 내 QR코드 렌더링 제거 (뷰어 최적화)

## Goal
현재 공유 링크에서 불필요하게 표시되어 직관성을 저하시키는 'QR 코드' 관련 렌더링 요소를 제거하고 비직관적 코드를 정리합니다. 기존에는 로컬 기기 간 복사가 어려워 QR로 브릿지했으나, URL 공유 체제로 전환됨에 따라 해당 요소가 더 이상 필요하지 않습니다.

## Plan
- `viewer.js` 및 연관 모듈(예: `export.js` 또는 `qr.js`)에서 QR 코드 생성 관련 로직(qrcode.js 등) 및 UI 인젝션 코드를 식별 후 삭제 또는 주석 처리.
- 공유 모드 및 단일 문서(뷰어) 진입 시 QR코드가 노출되던 화면 레이아웃에서 잉여 여백(Margin) 제거 및 스타일(CSS) 최적화.
