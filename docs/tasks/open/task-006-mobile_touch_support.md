# Task 006: 모바일 및 태블릿 환경 터치 이벤트 지원 (모바일 최적화)

## Goal
현재 마우스 이벤트 중심의 드로잉 로직을 터치 이벤트까지 호환하도록 확장하여 태블릿/모바일 크로스 플랫폼 지원을 보완합니다.

## Plan
- 캔버스 내 등록된 기존 `mousedown`, `mousemove`, `mouseup` 이벤트에 대응하는 `touchstart`, `touchmove`, `touchend` 이벤트 리스너 병렬 결합.
- 모바일 브라우저의 전매특허인 스크롤 방해 현상(Touch action 방지 처리) 해결.
- (선택) 반응형(Responsive) CSS 보완 적용.
