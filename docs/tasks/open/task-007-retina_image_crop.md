# Task 007: 고해상도(Retina) 디스플레이 번짐 완화 및 크롭(Crop)

## Goal
현재 캔버스의 고질병인 고해상도 환경의 블러(Blur) 현상을 해결하고, 사용자가 붙여넣기 한 이미지에서 부분만 재사용할 수 있는 간단한 크롭 에디터를 추가합니다.

## Plan
- `window.devicePixelRatio`를 Canvas 속성에 곱하여 논리 픽셀/물리 픽셀의 오차 보정 코드 주입.
- 이미지 내 사각형 영역만 DrawImage 시 추출할 수 있는 마스킹(Masking) 편집기 모달 구성.
