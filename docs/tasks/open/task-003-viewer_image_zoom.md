# Task 003: 공유 페이지(뷰어) 이미지 원본 보기 (확대 줌) 기능

## Goal
웹 공유 링크(뷰어 모드)에서 사용자가 이미지를 단일 클릭했을 때, 별도의 모달(Lightbox)이나 확대된 레이어 위젯을 통해 고해상도의 원본 크기로 이미지를 감상할 수 있는 기능을 제공합니다.

## Research & 쟁점
- 현재 재봉툴의 캔버스는 이미지 사이즈를 최적화하여 출력하고 있어, 전체적인 흐름을 보기에는 좋지만 세밀한 부분(디자인 디테일, 텍스트 등)을 공유받은 제3자가 확인하기가 어렵다는 단점이 있습니다.
- 이미지 확대 액션을 위해서는 해당 이미지의 원본 소스(DB 스토리지 URL 또는 Base64 데이터)에 접근할 수 있어야 합니다. (Task 001의 서버 데이터 모델과 구조를 공유함)

## Plan
1. **이벤트 리스너 부착:** 뷰어 모드(`readonly` 또는 권한 없음 상태) 렌더링 시, 각 이미지 박스 노드에 `click` 이벤트를 삽입합니다.
2. **Lightbox 모달 컴포넌트 개발:** 
   - 최상위 화면을 검은 반투명(Dimmed) 처리하는 오버레이 및 중앙에 확대된 이미지를 띄우는 별도의 UI(`div`) 요소를 추가합니다.
   - 바깥 여백(Dimmed Area)이나 우측 상단 X 버튼을 누르면 닫히는 로직 구현.
3. **스타일(CSS) 적용:** 부드러운 전환(Transition) 효과 적용.
