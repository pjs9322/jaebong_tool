# 🚀 재봉툴 PRO 로드맵 (Roadmap)

재봉툴 PRO의 전체적인 방향성과 마일스톤, 장기 목표를 관리하는 문서입니다.
현재 재봉툴 프로젝트는 Client-side 단일 어플리케이션으로서 바닐라 JS(Vanilla JS)와 FSM 패턴을 훌륭하게 아울러 핵심 기능을 매우 밀도 있게 구현했습니다.
앞으로는 실제 상용화 및 사용자 경험(UX) 관점에서 "기기 호환(터치), 데이터 보존(로컬 저장소), 실시간 공유(백엔드 API)" 과제들을 수행하여 SaaS 형태의 서비스로 운영하기에 손색이 없는 퀄리티로 고도화하는 것을 목표로 합니다.

## 🎯 주요 마일스톤 (v14 ~)

1. **사용성 및 데이터 파이프라인 구축 (통합 마일스톤)**
   - [보류] 모바일 및 태블릿 환경 터치 이벤트 지원 (모바일 최적화)
   - 기기 식별(UUID) 기반 실시간 동기화 연동 (자동 저장 & 데이터 수집)
   - 수집 데이터를 기반으로 한 공유 권한 부여 및 뷰어 아키텍처 분기
   - 실행 취소 (Undo / Redo) 기능 도입

2. **실 협업 환경 구축**
   - 웹소켓 기반 실시간 뷰어 및 채팅 시스템 확장 (장기)
   - 비직관적 렌더링 요소 (QR코드) 비노출 제거 보수
   - 공유 페이지(뷰어) 내 이미지 원본 보기(클릭 시 확대) 기능 추가
   - [Post-MVP] 로컬 UUID 기반의 다중 문서 생성 및 대시보드(작업 목록) 화면 구축

3. **엔터프라이즈급 품질 고도화**
   - [보류] 고해상도(Retina) 디스플레이 번짐 현상 개선
   - [보류] 간편 이미지 크롭(Crop) 모드 추가
